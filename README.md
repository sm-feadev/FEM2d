# Project Title

A 2d fem solver for well defined geometries

## Description

A finite element solver for 2d plate with cutout problems with MITCS6 element formulation

## Getting Started

### Dependencies

* gmsh - mesher
* numpy, matplotlib, scipy, meshio
* paraview - post processor

### Installing

* pip install <see above>

### Executing program

* python mitcs6.py
